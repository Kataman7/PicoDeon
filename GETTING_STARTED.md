# Tutoriel – Utiliser le Picodeon (clavier MIDI)

## 1. Connexion à l’ordinateur

Branche le **Picodeon** à ton ordinateur via USB.  
Sur **Windows**, un bip de connexion confirme qu’il est reconnu.  
Sur **macOS**, aucun son particulier, mais le clavier est détecté automatiquement.

## 2. Test rapide du Picodeon

Pour vérifier que tout fonctionne, ouvre :  
https://dotpiano.com/

Le Picodeon est reconnu directement et chaque touche doit produire un son.

## 3. Utilisation sur téléphone Android

<!-- Image : App Piano Synthétiseur de musique -->
![App Piano Synthétiseur de musique](https://github.com/Kataman7/PicoDeon/blob/main/img/screen4.png)

Installe l’application **Piano Synthétiseur de musique** :  
https://play.google.com/store/apps/details?id=com.flexbyte.pianosynth

Ouvre ensuite les **paramètres** de l’application.  

<!-- Image : application + icône paramètres -->
![Paramètres de l'application](https://github.com/Kataman7/PicoDeon/blob/main/img/screen1.png)

Va dans l’onglet **MIDI**.  

<!-- Image : onglet MIDI -->
![Onglet MIDI](https://github.com/Kataman7/PicoDeon/blob/main/img/screen2.png)

Sélectionne **Picodeon** comme périphérique MIDI.  

<!-- Image : Picodeon sélectionné -->
![Picodeon sélectionné](https://github.com/Kataman7/PicoDeon/blob/main/img/screen3.png)

Une fois la configuration terminée, tu peux jouer immédiatement sur ton téléphone 🎶
